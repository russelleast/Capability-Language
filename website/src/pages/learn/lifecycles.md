# Lifecycles

A lifecycle describes progression through states.

Local lifecycles belong to the capability that declares them. Supervising lifecycles coordinate multiple contributor capabilities and require explicit sources for contributor outcomes or events.

Lifecycle steps can wait for events, require decisions from actors, define deadlines, and move to terminal states.
